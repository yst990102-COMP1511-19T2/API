We use loops in C to do things multiple times. Here is an example of a loop that prints all the numbers from 1 to 17 on a new line in ascending order:


//initialise counter to 1
int counter = 1; 
// loop until not <= 17
while (counter <= 17) { 
    // print counter
    printf("%d\n", counter);
    // increment counter
    counter = counter + 1; 
}
